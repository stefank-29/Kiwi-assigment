# Installation

Clone repository:

```
git clone https://github.com/stefank-29/Kiwi-assigment.git
```

### Run project:

```
cd Kiwi-assigment/
```

Frontend:

```
cd frontend/

npm install

npm run dev
```

Backend:

```
cd backend/

npm install

npm start
```

Open in browser at address:

```
http://localhost:3000/
```

### Built with:

-   React (Next.js)
-   Node.js (Express)
